## 项目概况
本项目由前端项目`monitor-web`、后端项目`monitor-node`和sdk开发`monitor-sdk`三部分构成。
### 技术栈
#### 前端
- 前端框架：Vue3
- 开发语言：TypeScript/HTML/CSS
- 打包工具：Vite
- UI库：ELement plus
- 状态管理：Pinia
- 路由管理：VueRouter + vite-plugin-pages
